how to run test suites:
==========================
robot TestCases\ -->it will run all the programs inside TestCases folder
robot TestCases\Reg*.robot -->regular expression
robot TestCases\*.robot

how to run tests parallely using robotframework-pabot:
=======================================================
pre requisties:Install below package in cmd & PyCharm
CLI: pip install -U robotframework-pabot
pabot --processes 2 TestCases\*.robot

how to save results in results folder
======================================
pabot --processes 2 --outputdir Results TestCases\*.robot -->here in this command new folder called "Results" will be created and test files will be saved along with report.html

Create a file in project "run.bat" and add these below lines in bat file
cd  C:\Users\amuthaiya\PycharmProjects\pythonAutomationProject
pabot --processes 2 --outputdir Results TestCases\Reg*.robot TestCases\LoginTest.robot --> after inserting these two lines just right click run.bat file which is inside "pythonAutomationproject " and select "run as cmd script"
-->this will create a "Results" folder with RegisterationTest.robot and LoginTest.robot and thus  report.html will be created
